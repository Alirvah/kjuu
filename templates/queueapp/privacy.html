{% trans "Privacy Policy" %}

{% blocktrans %} This Privacy Policy explains how personal data is processed when using the kjuu web application for virtual queue management. {% endblocktrans %}

{% trans "1. Data controller" %}

{% blocktrans %} The controller is the natural person operating this application and responsible for personal data processing under Regulation (EU) 2016/679 (GDPR) and applicable Slovak legislation. {% endblocktrans %}

{% trans "2. Categories of processed data" %}

  • {% trans "Account data: username and password hash." %}
  • {% trans "Queue participation data: queue code, join time, call time, queue position, unique customer code." %}
  • {% trans "Optional note data submitted by users while waiting in queue." %}
  • {% trans "Technical metadata required for security and service operation (e.g., IP and request timestamps)." %}

{% trans "3. Purposes and legal bases" %}

  • {% trans "Service operation and account authentication (GDPR Art. 6(1)(b))." %}
  • {% trans "Security protection, abuse prevention, and service integrity (GDPR Art. 6(1)(f))." %}
  • {% trans "Compliance with legal obligations, if required (GDPR Art. 6(1)(c))." %}

{% trans "4. End-to-end encrypted notes" %}

{% blocktrans %} Optional queue notes are encrypted in the browser. The server stores encrypted payload only. Decryption requires a private key generated and stored on the client device. {% endblocktrans %}

{% trans "5. Data retention" %}

{% blocktrans %} Personal data is retained only for the period necessary to provide the service and protect system security. Queue participation records are removed when queue membership ends, and account data is deleted after account deletion, unless legal obligations require longer retention. {% endblocktrans %}

{% trans "6. Recipients and transfers" %}

{% blocktrans %} Data is not sold. Data may be processed by technical providers strictly for hosting and operation of the service. No intentional transfer to third countries is performed unless necessary for infrastructure operation with appropriate safeguards. {% endblocktrans %}

{% trans "7. Cookies and session" %}

{% blocktrans %} The application uses technically necessary session cookies to keep users authenticated and secure state-changing actions. No marketing cookies are used by default. {% endblocktrans %}

{% trans "8. Data subject rights" %}

  • {% trans "Right of access, rectification, erasure, restriction, and data portability where applicable." %}
  • {% trans "Right to object to processing based on legitimate interests." %}
  • {% trans "Right to lodge a complaint with the Office for Personal Data Protection of the Slovak Republic." %}
