# Kjuu 🧾  
A fully self-sufficient, secure, and real-time digital queue management SaaS built with Django.

## 🚀 What is QueueScan?

QueueScan is a zero-setup digital queueing system that allows businesses, clinics, events, and individuals to manage queues efficiently using a simple web interface and unique join codes. No apps to download, no hardware needed—just scan a code, join the queue, and wait your turn.

## 🧠 Features

- 🔒 Secure login with password or social login (OAuth)
- ⚡ Real-time queue updates
- 📱 Mobile-first UI, works on any browser
- 🎯 Join queues via unique code or QR scan
- 🧾 Auto-generated queue codes
- 🛡️ Rate limiting and brute-force protection
- 📊 Admin dashboard for queue owners
- 📦 Dockerized for easy deployment

## 🛠️ Tech Stack

- **Backend**: Django, Django REST Framework
- **Database**: PostgreSQL
- **Frontend**: HTML + TailwindCSS + Vanilla JS (or Alpine.js)
- **Containerization**: Docker + Docker Compose
- **Deployment**: Raspberry Pi
